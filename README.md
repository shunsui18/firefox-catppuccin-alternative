# **Installation Instructions:**
- Go to the [theme page](https://addons.mozilla.org/en-US/firefox/addon/catppuccin-mocha-alternative/?utm_source=addons.mozilla.org&utm_medium=referral&utm_content=search) [Firefox Browser ADD-ONS](https://addons.mozilla.org/en-US/firefox/) website.
